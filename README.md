# climate_change_on_sea_urchins

This repository integrates the supplementary material section of 
"Increased sensitivity of sea urchin larvae to metal toxicity as a consequence of the past two decades of Climate Change and Ocean Acidification in the Mediterranean Sea" by Davide Sartori, Guido Scatena, Cristina Andra Vrinceanu, Andrea Gaion. *Marine Pollution Bulletin* 194 (2023), 115274. https://doi.org/10.1016/j.marpolbul.2023.115274

In particular, this repository contains the [statistic analysis](analysis.ipynb) done on [data](data.csv).
